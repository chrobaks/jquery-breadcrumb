# jquery-bread-crumb
Simple Breadcrumb with Javascript and Jquery


### version

  - 0.0.1



### tech

  - Javascript 5.x
  - jQuery 1.x
  
  
  
### desciption  

Dynamic BreadCrumb for Pc, Tablet and Smartphone



### using

  - first run init method to set dom elements
  - run add methods to build breadcrumb path
